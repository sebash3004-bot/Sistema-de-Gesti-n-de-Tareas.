# To-Do Backend API

API REST desarrollada con Node.js y Express para gestionar tareas.

## Funcionalidades

- Crear tareas
- Listar tareas
- Actualizar tareas
- Eliminar tareas

## Ejecutar

npm install  
npx nodemon server.js  

## Endpoints

GET /api/tasks  
POST /api/tasks  
PUT /api/tasks/:id  
DELETE /api/tasks/:id
